## 📌 Project Overview

This project is an interactive Power BI dashboard built to analyze PhonePe transaction data across different states and categories in India. The dashboard provides meaningful business insights using DAX, Power Query, data modeling, and interactive visualizations.

---

## 📊 Project Information

| Category | Details |
|----------|---------|
| 📌 Project Type | Power BI Dashboard |
| 📊 Domain | Digital Payments Analytics |
| 🛠 Tools Used | Power BI, DAX, Power Query |
| 📂 Dataset | PhonePe Transaction Dataset |
| 📈 Visualizations | KPI Cards, Maps, Bar Charts, Line Charts, Tree Maps |
| 🎯 Objective | Analyze transaction trends and provide business insights |
